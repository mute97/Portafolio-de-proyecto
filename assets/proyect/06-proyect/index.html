# Deporte # Creatividad #Edicion

Descripcion del proyecto

Se diseñaron tarjetas de presentación para cada jugador del equipo de fútbol, con el objetivo de destacar su posición en el campo y ofrecer una imagen profesional que ayude a mejorar la identidad del equipo. Cada tarjeta incluye una foto de identificación del jugador, junto con su puesto de juego, brindando así una representación clara y personalizada de cada miembro del equipo. Para la creación de las tarjetas se utilizaron herramientas de edición gráfica avanzadas, las cuales permitieron integrar la imagen oficial del equipo y su logo de manera armónica. El diseño fue cuidadosamente adaptado para reflejar la esencia y los valores del equipo, utilizando sus colores representativos para asegurar una coherencia visual con la marca del equipo. El resultado es una tarjeta llamativa, profesional y fácilmente reconocible.
