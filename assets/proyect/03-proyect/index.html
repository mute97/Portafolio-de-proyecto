Galeria de Imagenes

#web #Diseño web #HTML

Descripcion del proyecto

En este proyecto, se desarrollará una galería de imágenes responsiva utilizando únicamente HTML y CSS, diseñada para ofrecer una experiencia visual atractiva y funcional en cualquier dispositivo. La galería estará estructurada para ajustarse automáticamente a diferentes tamaños de pantalla, desde teléfonos móviles hasta computadoras de escritorio, asegurando que las imágenes se visualicen de manera ordenada y profesional.
